# Project Instructions

This repository contains Jalal Stati's personal corporate portfolio.

## Hosting

- Repository: `jom006/jalalstati`
- Branch: `main`
- Hosting target: Cloudflare Pages
- Build command: `npm run build`
- Build output directory: `dist`

## Development

- Primary stack: React + Vite
- The site is static. Do not add a backend or contact form processor unless explicitly requested.
- Contact actions should use static links such as `mailto:`, `tel:`, and WhatsApp links.
- Keep credentials, tokens, FTP details, admin passwords, and private deployment information out of this repository.

## Content

- Primary public language: English.
- Portfolio positioning: PHP, Laravel, React, and React Native developer.
- Public project highlights include Jaybi on Google Play and B Entertained.
- Store local image assets under `public/assets`.

## Documentation

- If human-facing documentation is added later, place it under `docs/` as HTML.
- `AGENTS.md` is the root-level exception because automation tools read it from the repository root.
